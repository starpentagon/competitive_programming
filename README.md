# Libraries for Competitve Programming

# 数学系(Math)

## 素数判定

* Prefix: is-prime
* Description: 与えられた整数Nが素数かどうかを判定する
* 計算量: $O(\sqrt{N})$

## 約数列挙
* Prefix: enum-divisors
* Description: 与えられた整数Nの約数を列挙する
* 計算量: $O(\sqrt{N})$

## 最大公約数
* Prefix: gcm-func
* Description: 整数m, nの最大公約数を求める
* 計算量: $O(\log(\max\{m, n\}))$

## 最小公倍数
* Prefix: lcm-func
* Description: 整数m, nの最小公倍数を求める
* 計算量: $O(\log(\max\{m, n\}))$
